# PhishShield - Real-time URL Detector Extension

PhishShield is a browser extension that helps protect you from phishing attacks by analyzing URLs in real-time.

## Features

- Real-time URL scanning
- Blocks malicious websites
- User-friendly interface

## Installation

1.  Clone the repository.
2.  Open your browser's extension management page.
3.  Enable "Developer mode".
4.  Click "Load unpacked" and select the `extension` directory.

## Backend Setup

The backend is a Flask server that uses a machine learning model to classify URLs.

1.  Navigate to the `backend` directory.
2.  Install the required packages: `pip install -r requirements.txt`
3.  Run the server: `python app.py`
